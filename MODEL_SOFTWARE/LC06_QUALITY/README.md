# LC06_QUALITY

**Lifecycle Phase:** Quality

## Purpose

This directory contains software for QMS (Quality Management System) tools, process quality automation, audit management, nonconformance tracking, and quality gate validators.

## Contents

Software artifacts in this directory include:
- QMS software and tools
- Process quality automation
- Audit management systems
- Nonconformance management tools
- Quality gate validators
- Quality metrics collection and reporting
- Compliance checking utilities

## Naming Convention

All software artifacts follow the **v6.0 nomenclature standard** with:
- `MODEL=SW` token
- `PHASE=LC06` token
- `KNOT` binding (typically K06 for quality)
- Full compliance with controlled vocabulary (Section 10.2 of main README)

## Usage

Place software here when it:
- Supports quality management processes
- Implements quality gates and checks
- Manages nonconformances
- Automates audit processes
- Tracks quality metrics
