# APPROVAL_MINUTES — SIGNOFF (K04)

**Category:** SIGNOFF  
**KNOT:** K04 — interfaces / ICDs / integration contracts (multi-system coupling)  
**Owner (AoR):** STK_CM

---

## Purpose

This folder contains **minutes and formal records** of approval meetings and decisions supporting **cross-ATA integration clearance** under K04.

---

## Scope

- Approval meeting minutes
- Decision records (integration/coupling decisions)
- Review board minutes
- Approval committee records
- Attachments referenced by minutes (agendas, attendance, action lists), if governed as part of the record

---

## Usage

Place all approval meeting minutes in this directory following the **nomenclature v6.0**.

### Recommended filename pattern (minutes)

`00_AMPEL360_SPACET_Q10_PLUS_GENERATION_PR_00_<LC>_K04_STK_CM__<MeetingOrDecisionRef>_SIGNOFF_MINUTES_<Ixx-Ryy>_<STATUS>.md`

Examples:
- `00_AMPEL360_SPACET_Q10_PLUS_GENERATION_PR_00_LC01_K04_STK_CM__CCB_2025-12-20_SIGNOFF_MINUTES_I01-R01_ACTIVE.md`
- `00_AMPEL360_SPACET_Q10_PLUS_GENERATION_PR_00_LC01_K04_STK_CM__CrossATA_Clearance_ATA21-ATA24_SIGNOFF_MINUTES_I01-R01_ACTIVE.md`

### Minimal content expectations (suggested)

- Date/time, location/channel
- Chair + attendees (roles/AoRs)
- Agenda items
- Decisions (with decision IDs if used)
- Actions (owner + due date)
- References/links to impacted SSOT registers (CR Register, ICD Catalog, Impact Matrix)
- Approval statement (what was approved and the clearance scope)

---

## Cross-reference (SSOT)

Each minutes record should be logged in:
- `MODEL_PROCESS/REGISTRY/SIGNOFF_REGISTER/` (link to the minutes file + decision scope)
