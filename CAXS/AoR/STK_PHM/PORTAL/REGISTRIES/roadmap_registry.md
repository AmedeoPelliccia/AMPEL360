# STK_PHM — PORTAL Roadmap Registry (SSOT)

## RM-PHM-PORTAL-001 — TALF Pilot (PHM Tool Access & Licensing)
**Objective:** Provide license-aware, integrated access to PHM tooling with deterministic preflight and auditable events.

**Plan:**
- LC01/K01: Requirements + AoR mapping + catalogs skeleton (K01-T001..003)
- LC03/K03: Preflight enforcement gate (K03-T001)
- LC06/K06: Audit events + evidence linkage (K06-T001)

**Exit criteria:**
- Tool launchpad is available for PHM with launch channels and context mounts
- Task start blocked when entitlement/license missing (auditable)
- CI validates tool references and required licensing metadata
